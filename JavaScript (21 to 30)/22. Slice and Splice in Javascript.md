# _Slice and Splice in Javascript_


## _Slice_
- End value is exclusive
<b>

```javascript
var users = ["Ted", "Tim", "Ton", "Sam", "Sor", "Sod"];

console.log(users.slice(1,3));

console.log(users.slice(1));

// Output
// ["Tim", "Ton"]
// ["Tim", "Tom", "Sam", "Sor", "Sod"]
```

## _Splice_

```javascript
var users = ["Ted", "Tim", "Ton", "Sam", "Sor", "Sod"];

users.splice(1,2,"HI");
console.log(users);

// Output
// [ 'Ted', 'HI', 'Sam', 'Sor', 'Sod' ]
```
</b>
