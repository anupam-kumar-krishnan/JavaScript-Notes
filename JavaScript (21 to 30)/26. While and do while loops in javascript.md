# _While and do while loops in JavaScript_


## _While_
<b>

```javascript
const myStates = [
  "Rajasthan",
  "Delhi",
  "Assam",
  1947,
  "Tamil Nadu",
  "Maharashtra",
];

let i = 0;

while(i < myStates.length){
   console.log(myStates[i]);
   i++;
}

// Output        
// Rajasthan     
// Delhi        
// Assam
// 1947
// Tamil Nadu
// Maharashtra
```

## _do while_

```javascript
do {
   console.log(myStates[i]);
   i++;
} while(i < myStates.length);


// Output
// Rajasthan
// Delhi
// Assam
// 1947
// Tamil Nadu
// Maharashtra
```
</b>
